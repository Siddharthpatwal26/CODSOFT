# Titanic Survival Prediction

## Internship
CODSOFT Data Science Internship

## Problem Statement
Build a machine learning model to predict whether a passenger survived the Titanic disaster.

## Dataset
Titanic Dataset from Kaggle

## Project Structure
- data/ → Dataset file
- outputs/ → Generated graphs
- titanic_project.py → Main Python file
- model.pkl → Saved trained model
- requirements.txt → Required libraries

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model Used
Logistic Regression

## Accuracy
Approximately 75–85%

## How to Run

1. Install libraries:
   pip install -r requirements.txt

2. Run the project:
   python titanic_project.py
