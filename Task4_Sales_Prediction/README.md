# Sales Prediction Using Python

## Internship
CODSOFT Data Science Internship

## Problem Statement
Predict product sales based on advertising budget in TV, Radio, and Newspaper.

## Model Used
Linear Regression

## Technologies
- Python
- Pandas
- Seaborn
- Scikit-learn

## Evaluation Metrics
- MAE
- MSE
- R2 Score

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python sales_prediction.py
